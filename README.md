# CircuitAI - AI电路图辅助工具

## 项目简介

CircuitAI是一个AI驱动的电路图辅助工具，专注于电路图识别、解释、错误检查。

## 核心功能（MVP）

1. **电路图识别与解释**
   - 上传电路图（PNG/PDF/JPG）
   - AI识别元件（电阻、电容、芯片等）
   - 生成拓扑结构说明和原理解释

2. **BOM表自动生成**
   - 提取所有元件
   - 生成标准BOM表（Excel/CSV）
   - 支持批量导入供应链链接

3. **电路错误检测**
   - 检测电源接地问题
   - 检测芯片反向连接
   - 检测去耦电容缺失
   - 生成错误报告和修复建议

## 技术栈

- **前端**：React
- **后端**：FastAPI（Python）
- **AI模型**：Claude Opus 4-6 / GPT-4Vision
